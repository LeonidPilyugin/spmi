Installation
------------

First, clone git repository:

.. code-block:: console

    $ git clone https://github.com/LeonidPilyugin/spmi
    $ cd spmi

Then, create a virtual python environment:

.. code-block:: console

    $ python -m venv .venv

Install dependencies:

.. code-block:: console

    (.venv) $ pip install -r requirements.txt
